Matchs

Matchs are games made up of 15v15

Match maker

To create a game MM will pull 30 people that are in its queue to create a match.

The ideal match takes each tier makeing up a list of 3 top tiers, 5 mid tiers, 7 bottom tiers for a 3/5/7 line up.

Maps

Maps are picked based on the last one players have played, and then in some order that nobody understands.
